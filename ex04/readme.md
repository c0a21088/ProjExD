# 基本ゲーム概要
- ランダムに動く爆弾に当たらないように操作するゲーム
- 爆弾に当たると終了。
- 矢印キーで上下左右にこうかとんを操作できる。
- 爆弾、こうかとんは画面外に出る事は無く、爆弾は跳ね返る。
#  追加機能
- cボタンを押すとスキンの変更が出来る
- WASDキーで操作出来る
- 爆弾が跳ね返るたびに加速する
- 爆弾に当たると即終了ではなくウィンドウを出して結果を出す
